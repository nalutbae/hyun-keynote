우측 패널에서 관찰할 것

01

도구 호출 (Tool Call)

AI가 어떤 도구를 사용하는지, 어떤 매개변수를 전달하는지 실시간으로 표시됩니다.

02

도구 결과 (Tool Result)

도구가 반환한 데이터가 표시됩니다. 날씨 정보, 검색 결과, 파일 내용 등이 여기에 나타납니다.

03

추론 과정 (Reasoning)

AI가 왜 그 도구를 선택했는지, 결과를 어떻게 해석하는지 추론 과정이 표시됩니다.

04

최종 응답 (Response)

모든 과정을 거친 후, 사용자에게 전달되는 최종 답변입니다. 자연어로 정리되어 표시됩니다.

SOUL.md 편집하는 방법

01

Edit Soul 버튼

Hermes Desktop 좌측 사이드바 상단에 있는 'Edit Soul' 버튼을 클릭합니다. 텍스트 편집 창이 열립니다.

02

텍스트 작성

마크다운 형식으로 AI의 성격, 역할, 행동 원칙을 작성합니다. 저장하면 즉시 반영됩니다.

03

마크다운 문법

제목은 #, 목록은 -, 강조는 **로 감싸기. HTML 태그도 일부 지원합니다. 복잡한 문법은 몰라도 됩니다.

04

저장 후 확인

Save 버튼을 누르고, 새 대화를 시작하여 AI의 응답이 달라졌는지 확인합니다.

Save 후 확인 방법

01

저장

Edit Soul 편집창에서 Save 버튼을 클릭합니다. SOUL.md가 즉시 저장됩니다.

02

새 대화 시작

Cmd+N으로 새 대화를 시작합니다. 기존 대화에서는 SOUL.md가 반영되지 않습니다.

03

차이 확인

같은 질문을 해보고, SOUL.md 적용 전후 응답 차이를 비교합니다.

04

수정 반복

원하는 응답이 나올 때까지 SOUL.md를 수정하고, 새 대화에서 다시 테스트합니다.
