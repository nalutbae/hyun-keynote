Hermes Agent 설치 3단계

1

Python 설치

python.org에서 3.12 버전을 다운로드합니다. 설치 시 Add to PATH 체크는 필수입니다. 체크하지 않으면 터미널에서 Python을 찾지 못합니다.

2

Hermes Agent 설치

터미널(명령 프롬프트)에서 pip install hermes-agent을 입력합니다. pip은 Python 패키지 관리자로, 앱스토어 같은 역할을 합니다.

3

API 키 설정

프로젝트 폴더에 .env 파일을 만들고 API 키를 입력합니다. 금고 비밀번호를 별도 보관함에 넣는 것과 같은 원리입니다.

AI가 만든 코드 해부: 기초 개념

import

from datetime import datetime

라이브러리에서 날짜/시간 기능을 가져온다. 마트에서 만두피를 사오는 것과 같다. 직접 만들지 않고 이미 만들어진 도구를 가져와 쓴다.

f-string

f"현재 시간: {datetime.now()}"

문자열 안에 변수를 넣는 방식. { } 안의 값을 계산해서 문자열에 끼워 넣는다. 엑셀의 & 연결 연산자와 비슷한 역할을 한다.

1-4-2. 첫 번째 실습07 / 12

코드 해부 1 · 30:00-36:00

"AI가 만든 코드를 한 줄씩 해부해봅시다. 이게 오늘 가장 중요한 활동입니다."

  • import: "첫 번째 줄의 'from datetime import datetime'은 무슨 뜻일까요? 아까 배운 라이브러리 개념이 여기 나옵니다. datetime이라는 라이브러리에서 날짜/시간 기능만 가져오라는 뜻입니다. 만두피를 직접 만들지 않고 마트에서 사오는 것과 같죠."
  • f-string: "f"현재 시간: {datetime.now()}"에서 f가 앞에 붙은 것을 보세요. 이것을 f-string이라고 합니다. 문자열 안에 { } 중괄호로 변수나 계산식을 넣을 수 있습니다. 엑셀에서 =A1&"년" 처럼 텍스트를 연결하는 것과 같은 원리입니다."

"여기서 포인트는, 우리가 코드를 직접 짠 게 아니라는 것입니다. 하지만 AI가 만든 코드를 해부하면서 파이썬 문법을 자연스럽게 배우고 있습니다."

에이전틱 코딩 · 1회차 · KEYNOTE 1-4-2

AI가 만든 코드 해부: 제어문

for

for i in range(1, 11):

1부터 10까지 반복하는 for 반복문. range(1, 11)은 1 이상 11 미만의 숫자를 생성한다. 11은 포함되지 않는다.

if

if i % 2 == 0:

i를 2로 나눈 나머지가 0이면 짝수. % 연산자는 나머지를 구한다. 조건이 참일 때만 다음 줄이 실행된다.

코드 해부 2 · 36:00-42:00

"이번에는 반복문과 조건문을 해부합니다."

  • for문: "'for i in range(1, 11):' — 이것은 1부터 10까지 숫자를 하나씩 꺼내서 i에 넣으라는 뜻입니다. range(1, 11)에서 11이 포함되지 않는다는 것을 반드시 기억하세요. 은행에서 1월~11월까지 데이터를 처리할 때도 이 방식을 씁니다."
  • if문: "'if i % 2 == 0:' — 이건 i를 2로 나눈 나머지가 0이면, 즉 짝수이면 다음 줄을 실행하라는 뜻입니다. % 기호는 나머지 연산자입니다. 5 % 2 = 1, 4 % 2 = 0 이렇게 계산됩니다."

비유: "for문은 은행에서 '모든 대출 건을 하나씩 검토하라'는 것과 같고, if문은 '그중에서 연체인 것만 골라라'는 것과 같습니다."

"이 두 가지 제어문만으로도 은행 업무의 상당 부분을 자동화할 수 있습니다."

이것이 에이전틱 코딩의 학습법:

  1. AI에게 무언가를 만들어달라고 요청한다
  2. 결과물을 받는다
  3. 결과물을 한 줄씩 해부하며 원리를 이해한다
  4. 이해한 것을 바탕으로 더 복잡한 것을 요청한다

"이 선순환이 계속되면, 여러분은 문법서 한 장 안 읽고도 파이썬을 쓸 수 있게 됩니다."

두 번째 실습에서 배울 것

함수

def classify_risk(grade):

입력을 받아 출력을 반환하는 코드 블록. 은행의 위험도 분류 로직도 이렇게 만든다. def는 define의 약자.

리스트

high_risk = []

여러 값을 순서대로 담는 상자. 고위험 대출자 목록을 담을 수 있다. [] 빈 대괄호로 빈 목록을 만든다.

파일 읽기

with open(...) as f:

CSV 파일을 안전하게 열고 닫기. 은행 데이터를 읽어올 때 필수적인 기능. with는 파일을 쓰고 나면 자동으로 닫아준다.
