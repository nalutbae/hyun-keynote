Step 2: API Key 발급

01

openrouter.ai 가입

브라우저에서 openrouter.ai를 열고 계정을 만듭니다. Google이나 GitHub 계정으로 간편 가입이 가능합니다.

02

API Key 생성

로그인 후 우측 상단 프로필 → Keys → Create Key를 클릭합니다. 키 이름은 'Hermes Desktop'으로 설정합니다.

03

Key 복사

생성된 키를 복사합니다. 키는 한 번만 표시되므로 안전한 곳에 보관하세요. 분실 시 새로 만들어야 합니다.

⚠️

주의사항

API Key는 비밀번호와 같습니다. 절대 공개场所(공개 채팅, SNS 등)에 공유하지 마세요. 과금 문제가 발생할 수 있습니다.

Hermes Desktop 3개 영역

01

왼쪽 사이드바

대화 목록, 설정(⚙️), 프로필 전환. 이전 대화를 찾거나 새 대화를 시작할 수 있습니다.

02

하단 채팅창

메시지 입력 영역. 여기에 질문이나 지시를 입력합니다. Enter로 전송, Shift+Enter로 줄바꿈.

03

오른쪽 패널

도구 실행 결과, 파일 편집 내역, 검색 결과 등 AI의 작업 과정을 실시간으로 보여줍니다.
